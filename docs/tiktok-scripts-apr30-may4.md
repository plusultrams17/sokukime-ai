## Day 8: 5/2(金) — 「前提設定」4ステップ先回り

### 台本（Vrewに貼り付ける）

```
「検討します」って言われたくないなら、
商談前にこの4ステップ。

1. 精一杯お話しします
2. 気に入らなければ断ってOK
3. 気に入ったらスタートしてください
4. よろしいですか？

ポイントは2番。
ここでYESもらえたら、3番で99%決まる。

先回りの練習、プロフから。
```

### TikTokキャプション

```
「検討します」が出るのは、先回りしてないから。
この4ステップで「考えます」を封じろ。

#営業マン #商談 #クロージング #営業テクニック #成約率 #先回り #営業ノウハウ
```
