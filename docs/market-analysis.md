### 3-1. 競合が取っていない空白地帯

```
┌─────────────────────────────────────────────────────────────┐
│                     競合空白マップ                           │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  【空白地帯】                                                │
│  ✅ 個人向け × AI × 低価格 × 体系的メソッド × セルフサーブ    │
│                                                             │
│  現状の競合は以下のどちらかに偏っている:                       │
│                                                             │
│  パターンA: 法人向け × AI × 高額 × カスタム                  │
│  → exaBase, amptalk, SAPI（個人は手が出ない）                │
│                                                             │
│  パターンB: 個人向け × 非AI × 低価格 × 汎用                  │
│  → YouTube, 書籍, ストアカ（アウトプット練習なし）             │
│                                                             │
│  パターンC: 法人向け × 非AI × 高額 × 対面                    │
│  → インソース, リクルートMS, セレブリックス（月1回の研修）      │
│                                                             │
│  → 「個人が月¥2,980で、体系的メソッドに基づくAIロープレを      │
│     毎日できる」サービスは存在しない                           │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

**空白地帯の根拠:**
1. **exaBase等は初期300万円+従量課金** → 個人では絶対に使えない
2. **Reflect AIは個人無料あり** → ただし営業特化ではなく汎用。独自メソッドなし
3. **YouTube/書籍は無料〜安価** → しかしアウトプット練習は不可能
4. **従来研修は月1回** → 型が身につくには頻度が圧倒的に足りない

## 8. 結論

### 市場機会

日本のAI営業ロープレ市場は「法人向け高額サービス」に偏っており、**個人営業マンが手軽に使える AI ロープレツールは事実上の空白地帯**。810万人の営業職人口に対して、月¥2,980で体系的メソッドに基づくAIロープレを提供するサービスは存在しない。

### 勝ち筋

1. **価格破壊**: 法人向けAIロープレの1/100以下のコストで同等以上の練習体験
2. **メソッドの独自性**: 1,600件の実商談から体系化した5ステップは簡単にコピーできない
3. **セルフサーブ**: 登録不要で即座に体験 → 営業マンの「今すぐ練習したい」ニーズに直結
4. **ボトムアップ採用**: 個人で使い始め → チームに広がる → 法人契約へ（PLG戦略）

### 最優先アクション（今週やること）

1. **X（Twitter）アカウントの開設・初投稿**
2. **noteで「4年半の商談をAIにした話」を公開**
3. **Product Hunt ローンチ日を設定（2週間後目安）**
