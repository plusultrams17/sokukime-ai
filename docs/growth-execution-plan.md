## 3. 勝てるポジショニング（3エージェント統合結論）

### 一言で言うと

> **「法人AIロープレの品質を、月2,980円で個人が使える。ChatGPTでは手に入らない"営業メソッド内蔵+スコアリング"付き」**

### ポジショニングマップ

```
                  高価格（法人向け）
                       │
    ナレッジワーク(月20万)  │  exaBase(初期300万)
    ailead             │  ACES Meet
                       │
  ──── メソッドなし ──────┼────── メソッドあり ────
                       │
    ChatGPT(無料)       │  ★ 即キメAI(月2,980円)
    汎用AI              │  堀口龍介書籍(1,500円/買切)
                       │
                  低価格（個人向け）
```

**即キメAIは右下の唯一のプレイヤー。「メソッドあり × 個人向け価格」のポジションを独占できる。**

### 差別化3本柱

1. **即決営業メソッド内蔵** — ChatGPTにはない体系化された22レッスン
2. **5カテゴリAIスコアリング** — 成長が数値で見える（ChatGPTにはない）
3. **法人品質を個人価格で** — exaBase 300万円の体験が月2,980円

## 8. エージェント定義ルール

各エージェントは `.claude/agents/` に定義ファイルを持つ。以下のルールに従う。

### 共通ルール（全エージェント）
1. **景表法遵守**: 根拠のない数値・効果を記載しない
2. **ファクトベース**: 推測ではなくデータ/コードに基づいて判断
3. **実行可能性**: 「やるべき」だけでなく「どうやるか」まで出力
4. **コスト意識**: API原価・運用コストを常に意識

### 個別ルール
- **devil-advocate**: 楽観禁止。全ての施策に反対意見を述べる義務。ピボット案を常に用意
- **market-researcher**: Web検索必須。数字で語る。「～と思われる」禁止
- **product-auditor**: ソースコード精読必須。ファイル名:行数で指摘。修正コードまで出す
- **growth-executor**: 分析結果を実行に変換。コード変更・ドキュメント作成を担当
