## 声のかけ方のコツ

1. **最初の5人は「頼みやすい人」から** — 親しい知人、後輩、仕事仲間
2. **「5分だけ」を強調** — ハードルを下げる
3. **「率直な感想」を求める** — 褒め言葉より改善点が欲しいことを伝える
4. **お返しを用意** — コーヒーおごる、ランチおごる等
5. **フォローアップは1回だけ** — 2回以上催促しない
6. **成功体験を共有** — 「〇〇さんが面白いって言ってくれた」と次の人に伝える

## 目標スケジュール

| 日 | アクション |
|----|-----------|
| 1日目 | 5人にメッセージ送信 |
| 2-3日目 | 返信があった人と日程調整 |
| 4日目 | 返信なしの人にフォローアップ |

| 8日目 | フィードバック整理・追加で5人に送信 |
