# ChatGPTで営業ロープレをやってみたけど、成約率が上がらない理由

「ChatGPTに断り文句の切り返しを練習させてみたけど、なんか違う…」

そう感じたことはありませんか？ChatGPTは確かに便利なツールですが、営業スキルを伸ばすための「ロープレ練習」に使おうとすると、どこか物足りなさを感じる場面が多いはずです。本記事では、ChatGPTを使った営業ロープレの具体的な方法から、その限界、そして本格的なスキルアップにつながる代替手段まで解説します。

---

## ChatGPTで営業ロープレをする方法

ChatGPTで営業ロープレを始めるのは比較的簡単です。以下のようにプロンプトを入力すれば、AIが顧客役を演じてくれます。

**プロンプト例**
```
あなたは保険の営業マンに対して「今は必要ない」と断る顧客を演じてください。私が営業トークをしますので、自然に反応してください。
```

このように設定すれば、ChatGPTが顧客として受け答えしてくれます。断り文句に対して切り返しを試したり、クロージングフレーズを練習したりすることはできます。

手軽に始められるため、「まずChatGPTで練習してみよう」と考える営業マンが増えています。

## ChatGPTでの営業ロープレ、3つの大きな限界

ところが、実際に使い込んでみると、以下の問題に直面します。

### 限界1：スコアや客観的な評価がない

ChatGPTとロープレをしても、「今の会話、実際に何点だったの？」がわかりません。

会話が終わったあと、ChatGPTに「今のトークを評価して」と聞けば感想を言ってもらえますが、それはあくまで文章による定性的なフィードバックです。

- 傾聴力は何点か
- クロージングのタイミングは適切だったか
- ヒアリングの深さは十分か

こうした観点を数値で可視化してくれる機能はありません。スポーツで言えば、練習はするけど記録を測定しない状態。成長の実感がつかみにくく、モチベーションの維持も難しくなります。

### 限界2：体系的な営業メソッドが組み込まれていない

ChatGPTは汎用AIです。営業の専門知識はある程度持っていますが、「成約率を上げるための体系的なプロセス」に沿ったトレーニングはできません。

たとえば、

- ヒアリングの深掘り方法
- 不安を払拭するための話法
- 成約への自然な誘導ステップ

といった段階的なスキルを習得するには、バラバラな会話練習では不十分です。体系的なメソッドに沿って繰り返し練習することで、はじめてスキルが定着します。

### 限界3：業種・シナリオの作り込みが浅い

保険営業、不動産営業、訪問販売など、業種によって顧客の反応パターンや断り文句は大きく異なります。

ChatGPTに「保険の顧客を演じて」と指示しても、「今は考えていません」「夫/妻に相談してから」など、表面的な断り文句を返してくるだけです。

実際の現場で遭遇する「なぜ断るのか」の深い動機まで踏み込んだリアルな反応は、プロンプトを工夫しても再現が難しいです。

- 傾聴力

数値で見えるため「今月はクロージングが弱い」「先月より傾聴力が上がった」という具体的な成長の把握が可能です。

### 違い2：成約メソッドに基づいた体系的な練習

成約コーチAIは「成約5ステップメソッド」という体系的なフレームワークに基づいて設計されています。ステップごとの練習をすることで、スキルを段階的に積み上げていく仕組みになっています。

ChatGPTのように「とりあえず会話練習」ではなく、「どのステップのどのスキルを鍛えるか」を意識した練習ができます。

### 違い3：業種特化シナリオ

保険営業・不動産営業・訪問販売など、業種別のリアルな断り文句を収録したシナリオで練習できます。「今は時期が悪い」「もっと安い商品があれば」など、現場で実際に遭遇する言葉に対する切り返し話法を練習することで、本番の商談で応用しやすくなります。
