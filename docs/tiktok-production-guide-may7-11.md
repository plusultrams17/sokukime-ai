## 共通設定

| 項目 | 設定値 |
|------|--------|
| アスペクト比 | 9:16（縦） |
| 解像度 | 1080 x 1920 |
| 尺 | 15秒以内（厳守） |
| 背景色 | 黒 #000000 |
| フォント | 太ゴシック |
| メイン文字色 | 白 #FFFFFF |
| キーワード色 | オレンジ #f97316 |
| 警告/ネガティブ色 | 赤 #ef4444 |
| 文字サイズ | 画面幅の70〜80%を占める大きさ |
| 文字位置 | 中央配置 |
| BGM音量 | 20〜30%（テロップの邪魔をしない程度） |
| AI画像・動画生成 | 使わない（クレジット温存） |

---

## Vrewでの制作手順（1本10〜15分）

```
1. Vrew起動 → 新規プロジェクト → 画面サイズ「9:16」
2. 「テキストで始める」を選択
3. 台本テキストをコピペ（tiktok-scripts-may7-11.md から）
4. Vrewが自動でスライド分割してくれる
5. 以下を手動調整:
   a. 背景色を全スライド黒に変更
   b. フォントを太ゴシックに統一
   c. 文字色を白に統一
   d. キーワード部分だけオレンジ(#f97316)に変更（下記参照）
   e. 各スライドの表示時間を調整（下記参照）
6. BGMを追加（Vrew内蔵の音楽 or フリー素材）
7. プレビューで15秒以内に収まっているか確認
8. 書き出し → MP4 (1080x1920)
9. TikTokアプリで投稿
```

## 各動画のデザイン指示

### Day 10: 「商品を売るな、利点を売れ」

| スライド | 表示秒数 | オレンジにする箇所 | 演出 |
|---------|---------|------------------|------|
| 営業マンは商品を売るな。 | 0-2秒 | なし | ドンッと出現 |
| 利点を売れ。 | 2-4秒 | 「利点」 | スケールイン |
| 「24時間サポートが〜」 | 4-6秒 | なし、「売れない」を赤(#ef4444) | フェードイン |
| 「だから〜心配がなくなります」 | 6-10秒 | 「だから」 | スライドイン |
| たった一言「だから」〜 | 10-13秒 | 「だから」 | フェードイン |
| もっと知りたい人〜 | 13-15秒 | なし | フェードイン |

**BGM:** ビジネス系（落ち着いた、知的なテンポ）
**効果音:** スライド1でインパクト音（ドンッ）、スライド4で切り替え音

---

### Day 11: 「みなっっさん」の過半数テクニック

| スライド | 表示秒数 | オレンジにする箇所 | 演出 |
|---------|---------|------------------|------|
| クロージングで最強の〜 | 0-2秒 | なし | フェードイン |
| 「みなっっっさん〜」 | 2-5秒 | 全文オレンジ | ドンッと出現（大きめ文字） |
| ポイントは〜じゃない。 | 5-7秒 | なし | フェードイン |
| 「みなっっっっっさん」。 | 7-9秒 | 全文オレンジ（さらに大きく） | スケールイン + 効果音 |
| 81%の人は〜 | 9-12秒 | 「81%」 | フェードイン |
| 練習方法は〜 | 12-15秒 | なし | フェードイン |

**BGM:** ポップ系（テンポよく、明るい）
**効果音:** スライド2でインパクト音、スライド4で更に大きいインパクト音

---

### Day 12: 損は得の2倍強い（プロスペクト理論）

| スライド | 表示秒数 | オレンジにする箇所 | 演出 |
|---------|---------|------------------|------|
| 人が動く理由、知ってる？ | 0-2秒 | なし | フェードイン |
| 「損したくない」。 | 2-4秒 | 全文オレンジ | ドンッと出現 |
| 損の恐怖は〜2倍強い。 | 4-7秒 | 「2倍」 | 数字拡大エフェクト |
| だからトップ営業は〜 | 7-9秒 | なし | フェードイン |
| 「このままだと〜」 | 9-11秒 | 全文を赤(#ef4444) | スライドイン |
| ネガティブ80%〜 | 11-13秒 | 「80%」「20%」 | フェードイン |
| 練習したい人〜 | 13-15秒 | なし | フェードイン |

**BGM:** 緊張感のあるビジネス系（前半シリアス → 後半落ち着く）
**効果音:** スライド2でインパクト音、スライド3で効果音

#### Day 10転用（5/8）
```
「24時間サポートがあります」

これ、スペックを並べてるだけ。
お客さんは「で、自分にとって何がいいの？」としか思わない。

ここにたった一言「だから」を足すだけで変わる。

「24時間サポートがあります。
だから、トラブルで業務が止まる心配がなくなります。」

前半が理屈（セールスポイント）、
後半が感情（ベネフィット）。

「自分の生活がどう変わるか」。
```

#### Day 11転用（5/10）
```
クロージングで一番使えるテクニック、知ってますか？

「みなさん、こちら選ばれてます」

...これだと効果ゼロ。

「みなっっっっっさん、こちら選ばれてます！」

同じ言葉なのに、力の入れ方だけで
お客さんの反応がまったく変わる。

81.6%の人は他人のレビューを参考にしている（Gooリサーチ調べ）。
「みんなやってる」は人を動かす最強の後押し。

ポイントは3つ:
1. 「みなっっっさん」と力を込める
2. 身振り・ジェスチャーを必ず付ける
3. 商談中に繰り返し使う（序盤・中盤・終盤）

普通に言ったら社交辞令。
力を込めたら武器になる。

この過半数テクニックはAIで練習できます。
プロフィールのリンクから無料で5回お試しできます。

#クロージング #営業テクニック #社会的証明 #商談 #営業マン
```

#### Day 12転用（5/12）
```
人が動く理由は「得したい」じゃない。

「損したくない」。

心理学（プロスペクト理論）によると、
損の恐怖は得の喜びの2倍強い。

だからトップ営業は
メリットを語る前にリスクを語る。

「この商品のメリットは〜」→ 反応薄い
「このままだと○○できなくなります」→ 感情が動く

ただし注意点がある。
「○○様に限って当てはまらないですが」と
必ず敬意を添えること。

脅しじゃなくて、気づかせる。
ここを間違えると信頼を失う。

理想の配分はネガティブ80%、ポジティブ20%。
メリットだけで攻める営業は片腕で戦ってるのと同じ。

この心理テクニックはAIで練習できます。
プロフィールのリンクから無料で5回お試しできます。

#プロスペクト理論 #営業心理学 #クロージング #営業テクニック #営業マン
```
