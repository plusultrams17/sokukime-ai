## 投稿テンプレート（7パターン）

### 1. 切り返しフレーズ系（最も反応率が高い）
```
お客さんに「考えます」って言われたら、

❌「いつ頃ご検討いただけますか？」
⭕「そうですよね、大切なことですもんね。ちなみに何が一番気になりますか？」

ポイント：「考えます」の裏にある本音を引き出す。
9割の営業マンが「待ち」に入るところを「深掘り」に変える。

#営業 #切り返し #クロージング
```

### 2. 数字で語る営業の真実系
```
営業で成約率を上げるために一番大事なのは、

トーク力でも、商品知識でも、根性でもない。

「練習量」です。

プロ野球選手が素振りするように、
営業マンもロープレしないと上手くならない。

でも練習相手がいない——
それが営業業界最大の課題。

#営業 #ロープレ
```

### 3. Before/After ストーリー系
```
【塗装営業あるある】

現場で「高いね」って言われて黙ってしまう

→ 値引き提示

→ それでも「考えます」

→ 二度と連絡が来ない

これ、「高い」の裏にある本音を聞けてないだけ。

「高い」＝「価値がわからない」
「高い」＝「他社と比較したい」
「高い」＝「予算の相談をしたい」

反論処理を1つ覚えるだけで、成約率は変わる。

#塗装営業 #外壁塗装 #営業
```

### 4. ノウハウ図解系（画像付き推奨）
```
【成約5ステップ】

① アプローチ：信頼の土台を作る
② ヒアリング：本音を引き出す
③ プレゼン：価値で伝える（特徴じゃない）
④ クロージング：背中を押す
⑤ 反論処理：不安を解消する

この順番が大事。
いきなり③から始める人が多すぎる。

#営業の型 #営業研修
```

### 5. 共感・あるある系
```
営業の先輩に「ロープレしてください」って
頼むのが一番ハードル高い説。

・忙しそうで声かけられない
・ダメ出しが怖い
・毎回同じパターンになる

これ全部、AIが解決する時代になった。

#営業あるある
```
