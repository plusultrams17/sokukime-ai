## 投稿ルール（全プラットフォーム共通）

## TikTok 投稿計画

### 制作方法（Vrew）

#### 共通設定
- アスペクト比: **9:16（縦）**
- 尺: **15秒以内**
- 背景: **黒一色 (#000000)**
- フォント: **太ゴシック**
- 文字色: **白 (#FFFFFF)** / キーワード: **オレンジ (#f97316)**
- BGM: Vrewの音楽素材 or フリーBGM（ビジネス系、控えめ）
- **AI画像・動画生成は使わない**（クレジット温存）

#### Vrewでの手順（1本10〜15分）
```
1. Vrew起動 → 新規プロジェクト → 9:16 縦
2. 「テキストで始める」を選択
3. 台本テキストを貼り付ける
4. 背景色を黒に変更
5. テロップのスタイルを設定:
   - フォント: ゴシック系（太め）
   - サイズ: 画面幅の80%を占める大きさ
   - 位置: 中央
   - 色: 白（キーワードだけオレンジに手動変更）
6. 各テロップの表示時間を調整（台本の秒数に合わせる）
7. BGMを追加（音量は20〜30%程度）
8. プレビューで15秒以内に収まっているか確認
9. 書き出し（1080x1920, MP4）
10. TikTokアプリで投稿
```
