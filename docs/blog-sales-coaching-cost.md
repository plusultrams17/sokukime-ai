## 法人営業研修の費用相場

まず、一般的な法人向け営業研修の費用感を把握しておきましょう。

### 集合研修（外部講師招聘）

**相場：30〜100万円 / 回（1〜2日間）**

外部の営業コンサルタントや研修会社を招いて行う集合研修です。

- 参加人数：10〜30名程度
- 期間：1〜2日間
- 内容：営業理論の講義 + ロープレ + フィードバック

費用の内訳は講師費・資料作成費・会場費で構成されます。有名な講師・コンサルタントほど費用が高くなる傾向があります。

**1人当たりコスト**：30名参加で50万円の場合、1人あたり約1.7万円

### 継続型コーチング（個人コーチング）

**相場：月額5〜20万円 / 人**

プロのセールスコーチが1対1で継続的に指導するプログラムです。週1〜2回のセッションを通じて、個人の課題に合わせたフィードバックをもらえます。

最も効果が期待できる形式ですが、費用が高く、1人の営業マンに対してこの投資を継続するのは中小企業には難しいケースも多いです。

### eラーニング / オンライン研修

**相場：月額3,000〜30,000円 / 人 または 年間ライセンス**

営業の動画講座や演習問題が含まれるオンライン学習サービスです。インプット（知識習得）には向いていますが、実際の商談に近い「アウトプット練習」は含まれないものがほとんどです。

## 法人研修に潜む「見えないコスト」

費用のわかりやすい部分だけでなく、見落としがちなコストもあります。

### 機会損失コスト

1〜2日間の集合研修では、営業マンが商談に出られません。特に成績上位者は商談機会を失うことになります。

**例**：月売上200万円の営業マンが2日間参加 → 約13万円相当の機会損失

### 定着率の問題

集合研修の知識は、実践で使わないと急速に忘れます。研修後1週間でのスキル定着率は20〜30%程度と言われており（研修業界の一般的な傾向として）、定期的な反復練習がなければ投資効果が薄れやすいです。

### スケール問題

チームに新人が入るたびに研修を再実施すると、費用が積み上がります。10人採用するたびに50万円かかるなら、年間採用3回で150万円になります。

### 「効果」をどう考えるか

もちろん、費用だけで判断するのは適切ではありません。

法人研修のメリットは
- 人間の講師からのリアルタイムフィードバック

AIコーチングのメリットは

理想的なのは「法人研修で基礎を学び、AIコーチングで毎日定着させる」という組み合わせです。
