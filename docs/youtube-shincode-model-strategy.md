## 0. エグゼクティブサマリー

### しんコードモデルの核心

YouTubeは広告収入を稼ぐ場ではなく「無料の営業マン」。
収益比率 AdSense : Udemy : オンラインスクール = 1 : 6 : 3。
真の収益は有料プロダクト（Udemyコース + ShinCode_Camp月額2,500円）から生まれる。

### 成約コーチAIへの転用

- YouTube無料動画（営業ノウハウ + AIロープレ実演）→ SaaSサブスク課金
- しんコードの「ハンズオンチュートリアル」= 営業の「実演ロープレ動画」
- 制作コストはプログラミング動画より低い（コーディング不要）
- 競合YouTuberがほぼ不在（AI営業ロープレ実演は日本で未開拓）

## 4. 動画制作フロー（ソロプレナー向け）

### 必要機材
- PC（既存）+ マイク（2,000-5,000円）
- 画面録画: OBS Studio（無料）
- 編集: CapCut（無料）or Vrew
- サムネイル: Canva（無料）

### 1本あたりの制作工数

| 工程 | 所要時間 |
|------|---------|
| テーマ選定・構成 | 30分 |
| 台本作成 | 45分 |
| 撮影（画面録画+音声） | 30-60分 |
| 編集 | 60-90分 |
| サムネイル（Canva） | 15分 |
| アップロード・概要欄・タグ | 15分 |
| **合計** | **3-4時間/本** |

### 投稿頻度
- 通常動画: 週2本
- Shorts: 週1-3本（通常動画のハイライト切り出し）
- 週あたり制作時間: 最大8時間

## 7. マルチチャネル連動

### YouTube x X（Twitter）
- 動画のハイライト30秒をXにネイティブ投稿
- リプライでYouTubeフルバージョンURLを追加
- 動画台本から「一言パワーフレーズ」をテキスト投稿に転用

### YouTube x SEO記事
- 動画を文字起こしてブログ記事に転用（1コンテンツ2メディア）
- YouTube概要欄 → ブログ、ブログ → YouTube の相互リンク

### YouTube x Instagram/TikTok
- YouTube Shortsを先に制作 → 同素材をReels/TikTokに転用（追加工数ほぼゼロ）

## 8. リスクと対策

| リスク | 対策 |
|--------|------|
| YouTube制作に時間を取られすぎる | 週2本上限、制作時間8h/週以内 |
| 登録者数に一喜一憂 | 追うべき指標はLP訪問数と無料登録数 |
| 「ノウハウ発信者」止まり | 毎動画の概要欄にSaaS URL + CTA必須 |
| 景表法リスク | 「確実に上がる」等のNG表現を回避 |
