# TikTok動画 制作ガイド（Vrew）

> このファイルは制作方法・デザイン指示をまとめたもの。
> 台本は `tiktok-scripts-apr16-27.md` を参照。

## 共通設定

| 項目 | 設定値 |
|------|--------|
| アスペクト比 | 9:16（縦） |
| 解像度 | 1080 x 1920 |
| 尺 | 15秒以内（厳守） |
| 背景色 | 黒 #000000 |
| フォント | 太ゴシック |
| メイン文字色 | 白 #FFFFFF |
| キーワード色 | オレンジ #f97316 |
| 警告/ネガティブ色 | 赤 #ef4444 |
| 文字サイズ | 画面幅の70〜80%を占める大きさ |
| 文字位置 | 中央配置 |
| BGM音量 | 20〜30%（テロップの邪魔をしない程度） |
| AI画像・動画生成 | 使わない（クレジット温存） |

---

## Vrewでの制作手順（1本10〜15分）

```
1. Vrew起動 → 新規プロジェクト → 画面サイズ「9:16」
2. 「テキストで始める」を選択
3. 台本テキストをコピペ（tiktok-scripts-apr16-27.md から）
4. Vrewが自動でスライド分割してくれる
5. 以下を手動調整:
   a. 背景色を全スライド黒に変更
   b. フォントを太ゴシックに統一
   c. 文字色を白に統一
   d. キーワード部分だけオレンジ(#f97316)に変更（下記参照）
   e. 各スライドの表示時間を調整（下記参照）
6. BGMを追加（Vrew内蔵の音楽 or フリー素材）
7. プレビューで15秒以内に収まっているか確認
8. 書き出し → MP4 (1080x1920)
9. TikTokアプリで投稿
```

## 各動画のデザイン指示

### Day 1: 「検討します」の切り返し

| スライド | 表示秒数 | オレンジにする箇所 | 演出 |
|---------|---------|------------------|------|
| 「検討します」イコール〜 | 0-3秒 | 「検討します」 | スケールイン（ドンッと出現） |
| でも、この一言で〜 | 3-5秒 | なし | フェードイン |
| 「ありがとうございます〜 | 5-10秒 | 「何を検討されますか？」 | スライドイン or 1文字ずつ |
| 本音が出てくる〜 | 10-13秒 | なし | フェードイン |
| これ練習したい人〜 | 13-15秒 | なし | フェードイン |

**BGM:** ビジネス系（落ち着いた、控えめ）
**効果音:** スライド1でインパクト音（ドンッ）

---

### Day 2: 営業マンの絶望あるある

| スライド | 表示秒数 | オレンジにする箇所 | 演出 |
|---------|---------|------------------|------|
| 営業マンなら〜 | 0-2秒 | なし | ドンッと出現 |
| 客「いい話ですね〜」〜 | 2-5秒 | 「いい話ですね〜」 | フェードイン |
| 客「検討しますね」〜 | 5-8秒 | なし | フェードイン + 静寂の効果音 |
| 客「また連絡します！」〜 | 8-11秒 | 「その連絡は来ない。」を赤(#ef4444) | ガーンの効果音 |
| でも切り返しを〜 | 11-15秒 | なし | フェードイン |

**BGM:** コメディ系 or ポップ系（軽いノリ）
**効果音:** スライド3で静寂音、スライド4でガーン音

### Instagramキャプション

#### Day 1転用（4/16）
```
「検討します」は断り文句じゃなかった。

営業を始めた頃、この言葉が出るたびに
「あ、ダメだったか...」って諦めてた。

でも切り返しを1つ覚えてから、
「検討します」が怖くなくなった。

「ありがとうございます。
 ちなみに、何を検討されますか？」

たったこの一言で、相手の本音が出てくる。
そこからが本当の商談。

この切り返し、AIで何回でも練習できます。
プロフィールのリンクから無料で5回お試しできます。

#営業マン #商談テクニック #クロージング #成約率アップ #営業練習
```

#### Day 2転用（4/18）
```
「また連絡しますね！」

この言葉を信じて待ってた新人時代の自分に教えたい。
その連絡は来ないよ。

でも「検討します」「また連絡します」は
断り文句じゃなくて、実は本音を引き出すチャンス。

切り返しの型を1つ覚えるだけで、
あの沈黙が怖くなくなる。

練習場所はプロフィールのリンクから。
無料で5回試せます。

#営業あるある #営業マン #商談 #仕事術 #ビジネススキル
```

#### Day 3転用（4/20）
```
「高いですね」って言われた時、
値引きしてしまうのは三流。

トップ営業はたった一言で
「価格の議論」を「価値の議論」に変える。

「何と比べて、ですか？」

「高い」は感覚。
比較対象を明確にするだけで、
商談の流れが変わる。

プロフィールのリンクから、
この切り返しをAIで練習できます。

#営業テクニック #価格交渉 #商談 #クロージング #営業マン
```

#### Day 4転用（4/23）
```
切り返し0回 → 成約率1.8%
切り返し3回 → 成約率68%

差、37倍。

ほとんどの営業マンが1回断られたら引き下がる。
理由はシンプルで「切り返しの練習をしたことがない」から。

スポーツと同じで、練習しないでできるわけがない。

AIが24時間練習相手になるアプリを作りました。
プロフィールのリンクから無料で5回試せます。

#成約率 #営業データ #営業練習 #ロープレ #営業マン
```

#### Day 5転用（4/25）
```
帰社後の「で、結果は？」

この一言が怖すぎて、
商談終わりの帰り道が憂鬱だった。

「検討するって言われました...」
「それ断られてるやん」

わかってる。わかってるけど言わないで。

でも気づいた。
報告が怖いのは「準備不足」が原因だった。

商談前10分、AIで練習するだけで
「今日はいけました」って報告できるようになった。

プロフィールのリンクから無料で体験できます。

#営業あるある #上司 #新卒営業 #社会人1年目 #営業マン
```

#### Day 6転用（4/27）
```
「今は必要ないです」

普通の営業マンはここで引き下がる。
でもトップ営業は1つだけ質問を追加する。

「ありがとうございます。
 ちなみに、いつ頃必要になりそうですか？」

「今は」と言った人は「いつかは」必要だと思っている。
その「いつか」を具体化するだけで、
次のアポが自然に生まれる。

この切り返しの練習方法はプロフィールから。

#営業テクニック #切り返し #アポ取り #商談 #成約率
```
