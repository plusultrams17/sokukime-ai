## 共通設定

| 項目 | 設定値 |
|------|--------|
| アスペクト比 | 9:16（縦） |
| 解像度 | 1080 x 1920 |
| 尺 | 15秒以内（厳守） |
| 背景色 | 黒 #000000 |
| フォント | 太ゴシック |
| メイン文字色 | 白 #FFFFFF |
| キーワード色 | オレンジ #f97316 |
| 警告/ネガティブ色 | 赤 #ef4444 |
| 文字サイズ | 画面幅の70〜80%を占める大きさ |
| 文字位置 | 中央配置 |
| BGM音量 | 20〜30%（テロップの邪魔をしない程度） |
| AI画像・動画生成 | 使わない（クレジット温存） |

---

## Vrewでの制作手順（1本10〜15分）

```
1. Vrew起動 → 新規プロジェクト → 画面サイズ「9:16」
2. 「テキストで始める」を選択
3. 台本テキストをコピペ（tiktok-scripts-apr30-may4.md から）
4. Vrewが自動でスライド分割してくれる
5. 以下を手動調整:
   a. 背景色を全スライド黒に変更
   b. フォントを太ゴシックに統一
   c. 文字色を白に統一
   d. キーワード部分だけオレンジ(#f97316)に変更（下記参照）
   e. 各スライドの表示時間を調整（下記参照）
6. BGMを追加（Vrew内蔵の音楽 or フリー素材）
7. プレビューで15秒以内に収まっているか確認
8. 書き出し → MP4 (1080x1920)
9. TikTokアプリで投稿
```

## 各動画のデザイン指示

### Day 7: 「2度褒め」の技術

| スライド | 表示秒数 | オレンジにする箇所 | 演出 |
|---------|---------|------------------|------|
| お客さんを褒めたら〜 | 0-3秒 | なし | フェードイン |
| でもトップ営業は〜 | 3-5秒 | 「もう1回褒める」 | スケールイン |
| 「いや、今まで100社〜 | 5-10秒 | 「ダントツですよ」 | スライドイン |
| 2度褒めると〜 | 10-13秒 | 「2度褒め」 | ドンッと出現 |
| この技術〜 | 13-15秒 | なし | フェードイン |

**BGM:** ビジネス系（温かみのあるテンポ）
**効果音:** スライド4「2度褒め」でインパクト音（ドンッ）

---

### Day 8: 「前提設定」4ステップ先回り

| スライド | 表示秒数 | オレンジにする箇所 | 演出 |
|---------|---------|------------------|------|
| 「検討します」って〜 | 0-2秒 | 「検討します」 | スケールイン |
| 1. 精一杯お話しします | 2-4秒 | 番号「1.」 | フェードイン |
| 2. 気に入らなければ〜 | 4-6秒 | 全文オレンジ | ドンッと出現 |
| 3. 気に入ったら〜 | 6-8秒 | 番号「3.」 | フェードイン |
| 4. よろしいですか？ | 8-9秒 | 番号「4.」 | フェードイン |
| ポイントは2番〜 | 9-13秒 | 「99%」 | 画面フラッシュ |
| 先回りの練習〜 | 13-15秒 | なし | フェードイン |

**BGM:** ビジネス系（テンポよく、リズミカル）
**効果音:** スライド3（ステップ2）でインパクト音、スライド6「99%」で効果音

### Day 9: 「具体的に」連発の深掘り質問

| スライド | 表示秒数 | オレンジにする箇所 | 演出 |

### Instagramキャプション

#### Day 7転用（5/1）
```
褒めたのに謙遜された。
そこで終わるのが普通の営業。

でもトップ営業は、もう1回同じ内容で褒める。

「いやいや、本当に。
今まで100社見てきましたけど、
ダントツですよ」

1回目の褒めは社交辞令に聞こえる。
でも2回目は「本気で言ってくれてるんだ」と伝わる。

しかも「100社中ダントツ」のように
比較対象を入れると説得力が跳ね上がる。

これが「2度褒め」の技術。
商談の空気がガラッと変わります。

この技術はAIロープレで何回でも練習できます。
プロフィールのリンクから無料で5回お試しできます。

#営業テクニック #商談 #コミュニケーション #褒め方 #営業マン
```

#### Day 8転用（5/3）
```
「検討します」って言われるのは、
商談中にお客さんが不安を抱えたまま進んだから。

だから商談の最初に「前提設定」をする。

1. 精一杯お話しします
2. 気に入らなければ断ってOK
3. 気に入ったらスタートしてください
4. よろしいですか？

ポイントは2番。
「断ってOK」と先に伝えることで、
お客さんの警戒心が一気に下がる。

そしてここで「はい」をもらえると、
3番の「気に入ったらスタート」にも
自然と「はい」が出る。

商談前にこの4ステップを入れるだけで、
「検討します」が激減します。

この先回り話法はAIで練習できます。
プロフィールのリンクから無料で5回お試しできます。

#クロージング #営業テクニック #商談 #成約率アップ #営業マン
```

#### Day 9転用（5/5）
```
ヒアリングで1回質問して終わっていませんか？

「具体的に」を2〜4回繰り返すだけで、
お客さんの本当の問題に辿り着ける。

例えば:
「売上が落ちてるんです」
→「具体的にどのくらいですか？」
→「月50万くらい下がりました」
→「具体的にいつからですか？」
→「去年の秋からですね...」

浅い問題が、深い問題に変わる。

そして最後に
「それ、どんな気分ですか？」
と聞くと、感情が出てくる。

感情が出たら、もう売り込みはいらない。
お客さん自身が「解決したい」と感じている状態。

注意点として「困りますよね？」のような
誘導尋問は信頼を壊すので使わないこと。

深掘り質問の練習はプロフィールのリンクから。
無料で5回お試しできます。

#ヒアリング #営業テクニック #深掘り質問 #商談 #営業マン
```
