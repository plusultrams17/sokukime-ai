### 3-2. TikTok 投稿戦略

#### 再現可能なバズフォーマット4つ

**A. 一言パワーフレーズ型（15秒）— 最も簡単**
```
[0-3秒] テロップ: 「99%の営業マンが知らない一言」
[3-8秒] テロップ: 「『検討します』→『何を検討されますか？』」
[8-12秒] テロップ: 「相手の本音が出てくる」
[12-15秒] テロップ: 「練習したい人→プロフのリンクから」
```

**B. Before/After型（30秒）**
```
[0-3秒] 数字の変化: 「成約率 8% → 31%」
[3-20秒] 3ステップ解説: AIで練習 → 弱点特定 → 集中練習
[20-30秒] AIデモ画面チラ見せ
```

**C. あるある共感型（15秒）**
```
[0-2秒] 「営業マンなら100%共感するやつ」
[2-10秒] シチュエーション再現
[10-15秒] 「でも練習してたら焦らなくなった」
```

**D. AIデモ実況型（45-90秒）— 差別化最強**
```
[0-3秒] 「AIに商談の練習相手になってもらった」
[3-30秒] 画面録画: AIとのロープレ実演
[30-45秒] スコア結果表示 + CTA
```

#### 投稿スケジュール: 週3本（月水金）
- 月21:00 — 一言パワーフレーズ型（15秒）
- 水21:00 — AIデモ実況型（45秒）
- 金19:00 — あるある共感型（15秒）

#### TikTokの鉄則
- **視聴完了率70%でバズのボーダーライン** → 15秒以下から始める
- 投稿後1時間の反応が全てを決める → 投稿後に他動画にいいね+コメント
- シリーズ化がアルゴリズムに有利
- 冒頭3秒で止めないと終わり

## 4. PLG（Product-Led Growth）最適化

### 4-1. Aha Moment の仮説

**「初回ロープレ後のスコア可視化」が転換点**

→ Free 5回の中で「自分の弱点が数字で見える」体験をさせることが最重要

### 4-2. 無料→有料転換を上げる施策

| 施策 | 実装工数 | 期待効果 |
|------|---------|---------|
| 残り回数の目立つ表示（4回目/5回目で特別メッセージ） | 2時間 | 緊急性の心理 |
| スコアシェア機能（OGP画像生成） | 8-12時間 | バイラルループの起点 |
| 「あなたの弱点を克服するレッスン」のサジェスト | 3時間 | 有料レッスンへの自然な導線 |
| 紹介プログラム（紹介者+5回、被紹介者+2回） | 15-20時間 | K-factor 0.3-0.5目標 |

### 4-3. バイラルループ設計

```
[ユーザーAがAIロープレ実施]
      ↓
[スコア結果画面に「友人に挑戦させる」シェアボタン]
      ↓
[OGP画像: 「営業力スコア○○点 / ランク: B」]
      ↓
[友人BがURLクリック → ゲスト体験5回]
      ↓
[友人Bが会員登録 → ユーザーAに+5回クレジット付与]
      ↓
[ユーザーAが無料クレジットを使い切る → 有料化動機UP]
```

**原価計算:** 紹介1件あたりAPI費用 = 約7.5円（5回 × ¥1.5/ロープレ）→ CACとして極めて安い

## 6. B2B vs B2C の判断

### 結論: **B2Cを先に。B2Bは Month 3以降**

| 観点 | B2C（個人営業マン） | B2B（法人営業チーム） |
|------|-------------------|--------------------|
| 意思決定速度 | 即決（個人判断） | 稟議→承認（数週間〜数ヶ月） |
| 必要な機能 | 現状で十分 | 管理画面、一括請求、SSO等が必要 |
| 導入事例 | 不要（個人判断） | 必須（まだない） |
| 価格帯 | ¥990-4,980/月 | ¥50,000-200,000/月 |
| LTV | 低い | 高い |

**Phase 1（今〜3ヶ月）**: B2C個人ユーザー50人獲得に集中
**Phase 2（3〜6ヶ月）**: B2C実績を武器にB2B法人向けページ作成・営業開始

## 8. やってはいけないこと（アンチパターン）

4エージェントが一致して警告した失敗パターン:

| # | やってはいけないこと | 理由 | 代替策 |
|---|---------------------|------|--------|
| 1 | **全チャネルに均等にリソースを分散** | 1人運用で5チャネル同時は不可能 | X:60% + SEO:20% + TikTok:15% + IG:5% |
| 2 | **フォロワー数を追いかける** | フォロワー数≠売上 | プロフィールクリック数とLP訪問数を追う |
| 3 | **CVRデータなしで広告を出す** | 穴の空いたバケツに水を注ぐ | 50ユーザー獲得後に検討 |
| 4 | **新機能追加で集客問題を解決しようとする** | プロダクトは十分。マーケが0% | 機能開発を止めてマーケに全振り |
| 5 | **RTキャンペーン（プロダクト無関係の景品）** | ターゲット外が増えるだけ | 価値提供型の投稿で自然増 |
| 6 | **外部リンクをXの本文に直貼り** | -30〜50%ペナルティ | リプライに分離 |
| 7 | **1日5投稿以上の量産** | 質低下 → 信頼スコア低下 | 1日1投稿を高品質に |
| 8 | **値下げで解決しようとする** | 価格の問題ではなく認知の問題 | 価値の伝え方を改善 |
| 9 | **LP最適化にこだわりすぎる** | n=2のデータでA/Bテストは無意味 | まず30+の外部セッションを獲得 |
| 10 | **宣伝投稿ばかり** | フォロー解除+ミュート | 有益7 : 共感2 : 宣伝1 |
