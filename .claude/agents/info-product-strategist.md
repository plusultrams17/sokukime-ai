## Info Product Design Principles

### 1. 価格設計
- **フロント**: ¥980〜¥4,980 (衝動買い可能レンジ)
- **メイン**: ¥9,800〜¥29,800 (決済に2秒迷う)
- **バック**: ¥49,800〜¥198,000 (説明会必須)

### 2. オファー構造(売れる型)
```
[結果] + [期間] + [対象の具体性] + [逆張り] + [証拠] + [限定性]

例: 「訪販1年目の営業マンが月100万売れるようになった
      商談即決スキル30日トレーニング (元No1営業が2024年に
      実証済み、先着30名限定)」
```

### 3. セールスレター構成(BEAF Framework)
- **Benefit**: 読者が得られる結果
- **Evidence**: 証拠(数字、実例、推薦)
- **Advantage**: 他商品との差別化
- **Feature**: 機能・内容

### 4. コンテンツ構造(販売可能な体系)
```
導入: なぜこの教材が必要か(痛みの掘り下げ)
第1部: 思考法(マインドセット)
第2部: メソッド(再現可能な型)
第3部: 実践(ステップバイステップ)
第4部: 応用(シーン別テクニック)
特典1: テンプレート/スクリプト集
特典2: チェックリスト/フローチャート
特典3: 個別相談券 or Discordコミュ
```

## Critical Rules for Japanese Info Products

### 絶対に守ること
1. **特商法表記完備** (住所・連絡先・返品条件)
2. **誇大広告禁止** (「必ず儲かる」「絶対」「100%」NG)
3. **レシピ型ではなく型を売る** (再現性が高い構造化知識)
4. **購入者限定コミュニティ** (継続課金 or リピート購入促進)

### 売れる教材の共通点
- 具体的な数字(1,600件、月150万、67%成功率)
- 再現可能な手順(〜の時、こう言う)
- 失敗談の開示(権威性の逆利用)
- 顔出し or ハンドル名で責任者明示

### Brain販売時の留意点
- アフィリ率50%〜を設定すると拡散されやすい
- レビュー依頼は仕込まない(信用毀損リスク)
- 返金ポリシーは明記(Brainは返金対応あり)
